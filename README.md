# The Room — subhamkanoi.com

A personal site for **Subham Kanoi** — Kolkata boy, founder of UR Hospitality (Urban Rasoi),
someone who feeds people for a living.

It isn't a portfolio. It's framed as a *room* you step into — built to carry three feelings:
**pride** in the work, **humility** about the roots, and the **emotion** of the city and the people.

## Structure
- `index.html` — the whole room, in one self-contained file (HTML + CSS + a little JS).
- `assets/img/subham.jpg` — the portrait that hangs at the entrance.
- `assets/img/kolkata-skyline.jpg` — hand-drawn Kolkata horizon (Howrah Bridge, Victoria Memorial, the tram).
- `assets/img/og.jpg` — social share preview.

## Design notes
- **Mood:** candlelit room — warm espresso background, brass lamplight that follows the cursor,
  film grain, soft vignette.
- **Type:** *Fraunces* (a warm, emotional serif) for the voice; *Instrument Sans* for the body.
- **Motion:** quiet scroll reveals; everything respects `prefers-reduced-motion`.
- Voice is first person, plain, unboastful.

## Run it
Just open `index.html` in a browser — no build step.

## Publish on GitHub Pages
1. Push to the `subhamkanoi` repo.
2. Settings → Pages → Deploy from branch → `main` / root.
3. (Optional) add a `CNAME` file with your custom domain.
